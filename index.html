Game Rules

Objective: Clear as many horizontal lines as possible.

  • Move: Use the Left and Right arrow keys to move the piece.
  • Rotate: Use the Up arrow key to rotate the piece.
  • Soft Drop: Use the Down arrow key to speed up the piece's descent.
  • Scoring: You get points for every line you clear. Clearing multiple lines at once gives you more points.
  • Game Over: The game ends when the pieces stack up to the top of the screen.
